# Projeto People Analytics - Análise de Performance e Treinamentos

## Objetivo
Analisar a eficiência dos programas de treinamento e a estrutura salarial de uma organização fictícia para apoiar decisões estratégicas de RH.

## Ferramentas Utilizadas
* Google Sheets / Excel
* Funções: VLOOKUP (PROCV), IF (SE) Aninhado, SUMIF, AVERAGEIF.
* Tabelas Dinâmicas e Dashboards interativos.

## Etapas do Projeto
1. **Limpeza de Dados:** Remoção de duplicatas e normalização de nomes/departamentos.
2. **Tratamento de Missing Values:** Preenchimento de ratings de performance ausentes com base na média do departamento.
3. **Análise de Custos:** Cálculo de compensação total integrando salários e custos de treinamento.
4. **Visualização:** Dashboard interativo com filtros por departamento e cargo.
